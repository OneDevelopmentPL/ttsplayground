<a name="english"></a>
## English

**TTS Playground** is a premium, web-based Text-to-Speech launcher designed for high-quality audio experimentation. It features a modern, ultra-smooth interface with glassmorphism aesthetics and real-time controls.

### ✨ Features
- **Native Browser TTS**: Uses your system's built-in voices for low-latency, high-quality synthesis.
- **Dynamic Controls**: Real-time sliders for **Speed** (Rate) and **Pitch**.
- **Ultra-Smooth UI**: Sophisticated animations and a dynamic "Stop" button that appears only when needed.
- **Progress Tracking**: A visual bar that tracks the speech progress word-by-word.
- **Multilingual UI**: Seamlessly toggle between English and Polish interfaces.
- **Persistent Settings**: Your preferences (speed, pitch, last voice) are saved automatically.

### 🚀 How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. *Alternatively*, run a local server:
   ```bash
   npx serve .
   ```

### 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS (CSS Grid, Flexbox, Custom Bezier Animations)
- **Logic**: Vanilla JavaScript (SpeechSynthesis API)
- **Icons**: Phosphor Icons

<a name="polski"></a>
## Polski

**TTS Playground** to zaawansowany launcher Text-to-Speech w przeglądarce, stworzony do wysokiej jakości eksperymentów z dźwiękiem. Charakteryzuje się nowoczesnym, ultrapłynnym interfejsem w estetyce „glassmorphism” oraz kontrolą parametrów w czasie rzeczywistym.

### ✨ Funkcje
- **Natywne TTS**: Wykorzystuje wbudowane głosy systemowe dla minimalnych opóźnień i najwyższej jakości.
- **Dynamiczna Kontrola**: Suwaki do regulacji **Prędkości** i **Tonu** głosu w czasie rzeczywistym.
- **Ultrapłynny interfejs**: Wyrafinowane animacje i dynamiczny przycisk „Stop”, który pojawia się tylko podczas czytania.
- **Śledzenie postępu**: Wizualny pasek postępu aktualizowany słowo po słowie.
- **Wielojęzyczność**: Błyskawiczne przełączanie między polskim a angielskim interfejsem.
- **Pamięć ustawień**: Twoje preferencje (prędkość, ton, ostatni głos) są zapisywane automatycznie.

### 🚀 Jak uruchomić
1. Sklonuj repozytorium.
2. Otwórz `index.html` w dowolnej nowoczesnej przeglądarce.
3. *Alternatywnie* uruchom lokalny serwer:
   ```bash
   npx serve .
   ```

### 🛠️ Technologie
- **Struktura**: Semantyczny HTML5
- **Stylizacja**: Vanilla CSS (Grid, Flexbox, niestandardowe krzywe Beziera)
- **Logika**: Vanilla JavaScript (SpeechSynthesis API)
- **Ikony**: Phosphor Icons
